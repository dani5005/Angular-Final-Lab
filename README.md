# Angular Task Manager

Aplicación SPA desarrollada con **Angular** para gestionar tareas (To-Do App).

Permite:

- Crear tareas
- Listar tareas
- Ver detalles
- Editar tareas
- Eliminar tareas
- Filtrar tareas por estado

La aplicación consume la API pública:

https://jsonplaceholder.typicode.com/todos

---

# Tecnologías utilizadas

- Angular
- TypeScript
- SCSS
- Angular Router
- Angular HttpClient

# Instalación

Clonar el repositorio:

git clone https://github.com/dani5005/Angular-Final-Lab.git

# Entrar en el proyecto:

cd Angular-Final-Lab

# Instalar dependencias:

npm install

# Ejecutar la aplicación

ng serve

# Abrir en el navegador:

http://localhost:4200
